Metodología de trabajo


Dadas las condiciones del proyecto, hemos trabajado de forma conjunta a través de GitHub, ayudándonos de algunos clientes Git, como Sourcetree o el cliente oficial de GitHub.

Para el noticiero, hemos implementado mediante XMLHttpRequest() la carga automatica de las noticias a través de un archivo XML, también hemos configurado el RSS para que obtenga las noticias según se vayan publicando.

¿No te lo crees?

Puedes revisar el noticiero o ver el código en GitHub

También hemos implementado un ERP (dolibarr) totalmente funcional, con 7 modulos configurados, para que nuestros futuros trabajadores puedan generar facturas, administrar los proveedores, llevar la contabilidad de la empresa de forma sencilla, etc. (y lo más importante, ¡llevarnos bien con hacienda!)
